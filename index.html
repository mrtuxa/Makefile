make ist ein Build automation Tool dass automatisch aus dem Vorhandenen Source
Code ein Executeable Program baut, indem es die eine bestimmte Datei (Makefile)
ausliest welches definiert was make auszufhren hat.
Mit dem Unix-Kommando make kann man auf komfortable
Weise ausfuhrbare Programme erstellen, besonders wenn
diese aus vielen Quelldateien erzeugt werden mussen. Vorteile
sind:
Es werden automatisch alle Quelldateien ubersetzt, die seit
dem letzten Erzeugen des ausfhrbaren Programms
verndert wurden.
Es werden aber auch nur diese neu ubersetzt, d.h.
Objektdateien, die zu unvernderten Quelldateien gehren,
werden nicht neu erzeugt.
